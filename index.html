  • The FIRST thing to do for every GRAMA is to DRAW GRAMATANA by either Georeferencing REVENUE MAPS or directly draw with help of surveyor.
  • Then check it on GOOGLE EARTH for its accuracy(if need correct it).
  • Make sure you map GRAMATANA correctly.
  • Every time you visit a place for NOC/NA
  • Save at least 4 corner points(GPS) ACCURATELY.
  • THEN Scan SURVEY SKETCH and Georeference it.
  • Draw POLYGON for every Survey No. with PERFECT ATTRIBUTE DATA.
  • Check it on GOOGLE EARTH for its accuracy(if need correct it).
  • NOW after making LAYOUT drawing, export a copy to .jpeg or .pdf
  • THEN Georeference .jpeg or .pdf with the help of POLYGON created during NOC/NA.
  • Draw POLYGON for every SITE, PARK, CA and ROAD using LINE with PERFECT ATTRIBUTE DATA.
  • Check it on GOOGLE EARTH for its accuracy(if need correct it).
  • IF gis file of NOC/NA is not available then, repeat the NOC/NA steps by visiting the place.
  • For LAYOUT drawings already approved scan a copy to .jpeg
  • Whenever you visit a nearby place of Layout Get GPS location of known edges in a layout.
  • THEN Georeference .jpeg
  • Draw POLYGON for every SITE, PARK, CA and ROAD using LINE with PERFECT ATTRIBUTE DATA.
  • Check it on GOOGLE EARTH for its accuracy(if need correct it).

FOR ATTRIBUTE DATA in NOC/NA

SET A STANDARD LIKE

  • 1st row is STATE
  • 2nd row is DISTRICT
  • 3rd row is TALUK
  • 4th row is HOBLI
  • 5th row is GRAMA
  • 6th row is R.SY.No(first part)
  • 7th row is R.SY.No(second part)
  • 8th row is Extent(Acre-Gunta.00)
  • 9th row is NOC Purpose
  • 10th row is NOC given date
  • 11th row is Land Conversion Date.
